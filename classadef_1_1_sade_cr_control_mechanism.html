Detailed Description

SadeCrControlMechanism adjusts the object according to the procedure of the Cr value in the SaDE algorithm.

The difference between this and ControlMechanism:

  • select(): the function to record is inside function "mean".
  • update(): the function to record is inside function "mean".
Requirement
Take out parameters from Parameters:
  • type std::size_t and name "target_index" in generate(), select(), and update().
The configuration
SadeCrControlMechanism has no extra configurations.
See setup() for the details.
