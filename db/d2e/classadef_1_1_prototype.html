Detailed Description

Prototype is the abstract base class of those can change the implementation class dynamically and can configure its internal states through the configuration file.

Classes derived from Prototype are registered to PrototypeManager when calling init_adef().

The configuration
Prototype has extra configurations:
  • member
    • name: "classname"
    • value: string
See setup() for the details.
