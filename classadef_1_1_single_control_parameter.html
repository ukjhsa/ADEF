Detailed Description

template<typename T, typename Enable = void>
class adef::SingleControlParameter< T, Enable >

SingleControlParameter manages single object.

Template Parameters
TThe type of the object.
The configuration
SingleControlParameter has extra configurations:
  • member
    • name: "initial_value"
    • value: nothing or number configurations or object configurations which is the class derived from Function.
See setup() for the details.

void adef::SingleControlParameter< T, Enable >::setup ( const Configuration &  config,
const PrototypeManager &  pm 
)
inlineoverridevirtual

Set up the internal states.

If RealSingleControlParameter is used and has the following configuration:

  • initial_value: 0.5

its configuration should be

  • JSON configuration
    "ControlParameter" : {
    "classname" : "RealSingleControlParameter",
    "initial_value" : 0.5
    }

If RealSingleControlParameter is used and has the following configuration:

  • initial_value: RealNormalDisFunction of mean 0.0 and standard deviation 1.0

its configuration should be

  • JSON configuration
    "ControlParameter" : {
    "classname" : "RealSingleControlParameter",
    "initial_value" : {
    "classname" : "RealNormalDisFunction",
    "mean" : {
    "classname" : "RealConstantFunction",
    "value" : 0.0
    },
    "stddev" : {
    "classname" : "RealConstantFunction",
    "value" : 1.0
    }
    }
    }

If RealSingleControlParameter is used and has no extra configuration.
its configuration should be

  • JSON configuration
    "ControlParameter" : {
    "classname" : "RealSingleControlParameter"
    }
