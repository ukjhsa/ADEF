Detailed Description

template<typename T, typename Enable = void>
class adef::ControlRange< T, Enable >

ControlRange manages the range of the object.

Template Parameters
TThe type of the object. It is arithmetic type.

ControlRange defines the following interfaces:

  • is_valid(): check whether the object is in the range.

Now ADEF support many kinds of ControlRange:

  • IntegerControlRange
  • RealControlRange
  • IntegerControlControlRange
  • RealControlControlRange
Setting
  • set_lower_bound()
  • set_upper_bound()
The configuration
ControlRange has extra configurations:
  • member
    • name: "lower_bound"
    • value: double
  • member
    • name: "upper_bound"
    • value: double
See setup() for the details.
