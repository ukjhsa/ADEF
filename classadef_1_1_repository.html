Detailed Description

Repository contains informations of the algorithm.

The algorithm includes what evolution process is, what problem evaluates, what mutation and crossover use, and how environmental selection operate, etc.

Repository includes all evolutionary states used in the algorithm and some helper feature such as statistics and parameter storage.
