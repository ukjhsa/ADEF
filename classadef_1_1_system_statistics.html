Detailed Description

SystemStatistics handles the statistics of entire system.

SystemStatistics collects the data of each experiment, including

  1. the number of the fitness evaluations (FEs)
  2. the error
  3. the best fitness

then calculates the value of above data:

  1. the best value
  2. the worst value
  3. the mean value
  4. the standard deviation
  5. the success performance

by formula

  • success performance = (the mean of FEs for successful runs) * (the number of the total runs) / (the number of the successful runs)
