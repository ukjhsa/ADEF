Detailed Description

CurrentFitnessControlUpdate updates internal states by the fitness value of current individual..

Requirement
Take out parameters from Parameters:
  • type std::size_t and name "target_index"
record parameters into Function:
  • name "current"
The configuration
CurrentFitnessControlUpdate has no extra configurations.
