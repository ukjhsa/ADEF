Detailed Description

System is the top class to clients.

System contains one or many experiments and generates the analysis of all experiments.

Call System::run() to execute.

The configuration
System has extra configurations:
  • member
    • name: "name"
    • value: string
  • member
    • name: "SystemStatistics"
    • value: object configuration which represents SystemStatistics.
  • member
    • name: "Experiment"
    • value: array configuration
      • elements: object configuration which represents Experiment.
See setup() for the details.
See also
see Experiment and Repository for the contents.
