Detailed Description

ControlledObject encapsulates the object.

ControlledObject encapsulates the object for the use of creating and copying an object.

It supports the following types T:

  • T is a non-pointer and non-std::shared_ptr to arithmetic type or a class does NOT inherit Prototype.
    • e.g., ControlledObject<double>, ControlledObject<std::string>.
  • T is a std::shared_ptr to an abstract class derived from Prototype.
    • e.g., ControlledObject<std::shared_ptr<RealFunction>>.
  • T is a non-pointer or std::shared_ptr to a concrete class derived from Prototype.
    • e.g., ControlledObject<RealConstantFunction>.
