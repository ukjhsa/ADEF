Detailed Description

template<typename T, typename G = std::mt19937>
class adef::RandomSelectionFunction< T, G >

RandomSelectionFunction generate the object that is randomly selected from storage.

RandomSelectionFunction contains a variety of objects. When calling generate(), it randomly selects one from them.

RandomSelectionFunction has the following feature:

  • generate(): return the randomly selected object.
  • record(): do nothing.
  • update(): do nothing.

Now ADEF support many kinds of RandomSelectionFunction:

  • IntegerRandomSelectionFunction.
  • RealRandomSelectionFunction.
  • IntegerControlRandomSelectionFunction.
  • RealControlRandomSelectionFunction.
The configuration
ConstantFunction has extra configurations:
  • member:
    • name: "object"
    • value: array configurations
      • element: number configurations or object configurations depends on RandomSelectionFunction::Object
See setup() for the details.
