Detailed Description

Population contains total individuals.

The data structure of individuals are invisible to other classes. The ways to access individuals are

  • Access an individual in each time.
  • Apply a function to each individual.
The configuration
Population has extra configurations:
  • member
    • name: "population_size"
    • value: unsigned int
  • member
    • name: "Individual"
    • value: object configuration which represents Individual.
See setup() for the details.
