Detailed Description

template<typename T>
class adef::MultipleControlParameter< T, std::enable_if_t<!std::is_arithmetic< T >::value > >

MultipleControlParameter template specialization for non-arithmetic type.

Template Parameters
TThe type of the object is the non-arithmetic type.

This template specialization does NOT have the member configuration named "initial_value"

The configuration
MultipleControlParameter template specialization has extra configurations:
  • member
    • name: "number_of_objects"
    • value: unsigned int
See setup() for the details.
